  • Install Golang
  • https://go.dev/dl/
  • go version

Setup Visual Studio Code

  • Install the Go extension
    • Open VS Code → Extensions sidebar → search “Go” (by the Go Team at Google). Install it.
    • Install Go tools (VS Code will prompt you the first time you open a .go file).
    • gopls → Language server - Basically, gopls makes Go in VS Code feel like a full IDE instead of a text editor. ✅ IntelliSense / autocompletion ✅ Jump to definition (F12) ✅ Find all references ✅ Real-time linting & error checking ✅ Code formatting & renaming ✅ Inline documentation (hover tooltips) ✅ Workspace-wide symbol search
    • dlv → Debugger // Go uses Delve for debugging
    • goimports, gofumpt → Code formatting
    • staticcheck → Linting & static analysis
  • Debugging in VS Code
    • Open your main.go
    • Press F5 → Select “Go” environment.
    • VS Code will generate a launch.json (debug config)
    • Set breakpoints, step through code, inspect variables — just like with Python or Java
    • With VS Code + Go, You get autocomplete, linting, formatting, debugging, and testing out of the box
  • Format on save
    • "go.formatTool": "goimports" in settings.json
  • Install gotemplate-syntax plugin
