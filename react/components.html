What is Component?

  • Component is a reusable piece of code that defines a part of the user interface (UI);
  • It can be a button, form, header or any other part of the page that you can think of;
  • Each component manages its own state, structure, style, behaviour and returns JSX to render on the screen;

Characteristics of Components

  • Reusability - Components can be used multiple times in a page;
  • Encapsulation - Each component manages its own state and logic, keeping concerns separate;
  • Composition - Complex user interfaces are built by combining multiple simple components;
  • Props and State - Components use props (short for properties) to receive data from their parent components and state to manage their own internal data that can change over time;

Types of Components

  • Function based components - we can use simple function(rfc) or arrow function(rafc);
  • Class based components - we are not using them after v16;
  • Each component will have it's own js file; First letter of the file should be in capital;

Functional Component with Props

                    // They are basically JavaScript functions that take in props (input data) and return JSX (JavaScript Syntax Extension) elements.

function UserCard({ name, age }) {
  return (
    <div>
      <h2>{name}</h2>
      <p>Age: {age}</p>
    </div>
  );
}

function App() {
  return (
    <div>
      <UserCard name="Alice" age={25} />
      <UserCard name="Bob" age={30} />
    </div>
  );
}


Functional Component with state

import React, { useState } from "react";

function Counter() {
  // Declare a state variable "count" with initial value 0
  const [count, setCount] = useState(0);

  return (
    <div style={{ textAlign: "center", marginTop: "20px" }}>
      <h2>Counter: {count}</h2>
      <button onClick={() => setCount(count + 1)}>Increase</button>
      <button onClick={() => setCount(count - 1)}>Decrease</button>
      <button onClick={() => setCount(0)}>Reset</button>
    </div>
  );
}

export default Counter;

import React from "react";
import Counter from "./Counter";

function App() {
  return (
    <div>
      <h1>React State Example</h1>
      <Counter />
    </div>
  );
}

export default App;
