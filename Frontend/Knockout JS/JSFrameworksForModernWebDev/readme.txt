To run an example:
1. Install Node.js, if not already done
2. Run 'npm install' to install in JSFrameworksForModernWebDev folder to download all dependencies;
3. Each example contains index.js file that runs a simple Node.js web server;
4. To run an example, navigate to JSFrameworksForModernWebDev/example-000 directory and type 'node index.js'
5. Open your browser and navigate to http://localhost:8080
