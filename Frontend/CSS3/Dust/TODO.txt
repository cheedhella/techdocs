<style> tag, <link> tag

<style> 
	HTML style element can only be inserted in the document's head
	Attributes:
		type - text/css
		media - http://www.htmlquick.com/reference/types.html#media-descriptor
			Specifies the media for which the element's content has been designed. It can be a single media or a comma-separated list of medias. The default value is "screen".
		scoped - new in HTML5, http://www.w3schools.com/tags/att_style_scoped.asp , http://updates.html5rocks.com/2012/03/A-New-Experimental-Feature-style-scoped
		disabled - http://www.whatwg.org/specs/web-apps/current-work/multipage/semantics.html#the-style-element
		https://developer.mozilla.org/en-US/docs/HTML/Element/style
	Eg:
		<style type="text/css">
			p {font-size: 8pt; font-family: arial,helvetica}
			p.big {font-size: 12pt; font-family: arial,helvetica}
		</style>
	
		
	
<link>
	Another alternative for inserting style sheet information, and maybe more convenient because of the possibility of sharing it among many documents, is to place a reference to an external file using the HTML link tag.
