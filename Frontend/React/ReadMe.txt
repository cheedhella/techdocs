What is a Component?
    It is an independent and reusable bit of code, similar to javascript function;
    It helps in dividing the UI into smaller pieces;
    It takes any arbitary input(called props) and returns JSX or React elements describing what should appear on the screen;
    Each component will go into its own JavaScript file;
