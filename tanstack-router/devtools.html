What is Tanstack Router Devtools?
  • It is used to inspect the current route, loader data, matches, etc. live in your app.
  • Install: npm install @tanstack/router-devtools
import React from "react"; import { createRoot } from "react-dom/client"; import { RouterProvider } from "@tanstack/react-router"; import { router } from "./router"; // ✅ import devtools import { TanStackRouterDevtools } from "@tanstack/router-devtools"; createRoot(document.getElementById("root")!).render( {/* ✅ Add DevTools after RouterProvider */} ); Customize position (optional) The position prop controls where the DevTools panel floats: "bottom-right" (default) "bottom-left" "top-right" "top-left" Example: Usage Run your app: npm run dev Click the TanStack logo button in the corner → it opens the devtools panel. You’ll see: Current route tree & active matches Loader status (idle / loading / error) Data returned by loaders Router state This is especially helpful with your /posts loader and /posts/new action — you’ll see the loader refresh when router.invalidate() runs after creating a new post.
